## 3.X 预测器长度偏倚量化及其对蛋白组规模挖掘的影响

**背景与动机。** DPP-IV 抑制肽计算预测器的可靠性长期受质疑，其准确性被认为受"长度混杂"影响——正样本多为短肽、负样本多为长肽或全长蛋白（PepBenchmark、StackDPPIV 等多次将其列为开放问题）。然而，长度偏倚究竟在多大程度上被误读为"活性信号"、又如何系统性扭曲"蛋白组规模虚拟挖掘"的候选优先级，尚缺乏定量证据。本研究以 iDPPIV-SCM（Charoenkwan et al. 2020，已本地离线复现）为案例对象，首次对上述问题给出量化结果，并据此提出长度稳健的候选排序协议。

**方法。**
1. 在 iDPPIV-SCM 同源公开基准上复现**全局组成型 SCM**（得分 = Σ_a∈S P(a)，判定阈值 −1.148 经嵌套 5 折交叉验证优化），报告独立测试集 ACC 与 MCC。
2. 设定**琐碎长度基线**：肽长 ≤ T（T = 6/8/10/12）→ 判为 DPP-IV 抑制阳性，比较其 ACC/MCC 与 SCM。
3. 统计基准正/负样本的长度分布，并按长度分层（≤6 aa / >6 aa）对比 SCM 与长度基线的 ACC，定位偏倚来源。
4. 在毛竹候选短肽池（n = 4950，2–6 aa）上，计算 SCM 全局分、长度归一分（每残基平均）、独立 SI 风格概率与肽长的 Spearman 相关；并提出**长度稳健残差重排**（线性回归 SCM 分对长度后取残差再排名），比较原始 Top20 与稳健 Top20 的重叠，评估偏倚对排序的实际影响。

**结果。**

- 全局组成 SCM 独立测试 **ACC = 0.771（MCC = 0.555）**，与文献报道（≈0.797）接近；位置特异变体 ACC 更低（0.620）。
- 然而，一条**仅依赖肽长的琐碎基线**（len ≤ 10 → 阳性）即达 **ACC = 0.820（MCC = 0.648），高于 SCM 自身**。在长肽层（>6 aa）SCM 为 0.692、长度基线为 0.833；在短肽层（≤6 aa）SCM 仅 0.518、长度基线达 0.800——即 SCM 在两个长度层均不优于纯长度启发式。
- **根因**：基准正样本中位长度 **4**（mean 5.31，范围 2–18），负样本中位长度 **15**（mean 15.53，范围 5–75），长度分布严重混杂（表 X1）。
- 在毛竹候选池内，长度偏倚以"**更短者得分更高**"的形式延续：SCM 全局分 ~ 长度 ρ = **−0.188**，独立 SI 概率 ~ 长度 ρ = **−0.212**；而长度归一分（每残基平均）将相关降至 ρ = **−0.056**，证明长度归一可部分缓解该偏倚。
- **长度稳健残差重排**使原始 Top20 中 3 条被替换（重叠 17/20）；而本研究三个决赛候选 **LPPGP / APPSQ / APQIP** 经残差重排后排名不变（16→16、420→420、107→107），表明其优先级**对长度偏倚稳健**。

*表 X1. 基准集长度分布混杂与长度基线表现（思路① 核心证据）*

| 指标 | 数值 |
|---|---|
| 基准训练 / 测试 | 1063 / 266（正负各半） |
| 全局 SCM 独立 ACC / MCC | 0.771 / 0.555 |
| 纯长度基线（len≤10→阳）ACC / MCC | 0.820 / 0.648 |
| 正样本长度 median / mean | 4 / 5.31 |
| 负样本长度 median / mean | 15 / 15.53 |
| 候选池 SCM分~长度 ρ | −0.188 |
| 候选池 SI概率~长度 ρ | −0.212 |
| 候选池 长度归一分~长度 ρ | −0.056 |
| 残差重排后 Top20 重叠 | 17/20 |

**结论与对挖掘的意义。** iDPPIV-SCM 所报告的精度有相当部分源于长度混杂、而非真实的残基组成信号；在蛋白组规模挖掘中，若直接以 SCM 分排序，候选池会被系统性偏向极短肽。我们建议：（a）以**长度归一分或残差重排**作长度稳健排序；（b）在报告任何"高活性候选"时**显式披露长度分布并做分层验证**。这一量化结果从方法学层面解释了"单纯再筛一遍"难以产出可靠候选的原因，也为本研究 §3.7 的候选优先级结论提供了偏差校正依据——本研究的决赛候选经证明对长度偏倚稳健，其优先级不依赖于该混淆信号。
