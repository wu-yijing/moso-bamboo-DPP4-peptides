# 思路③ 转录组表达层（可食嫩笋表达证据）— 章节草稿

> 定位：生信方法论文《可复现离线框架》的第二支柱（基因组存在性 → 可食部位表达）。
> 本层把"候选是基因组编码、天然存在"（思路③ 本地部分）进一步升级为
> "候选源蛋白属于在**可食嫩笋**中高表达的功能家族"的定性证据，
> 以化解"253 条含非食物蛋白、食物转化性弱"的软肋。

---

## 3.x 可食嫩笋表达证据层（转录组桥接）

### 3.x.1 动机与边界
纯计算研究无法提供湿实验的"器官水平定量表达（TPM/FPKM）"。然而，已发表的毛竹
转录组资源为"候选源蛋白是否属于可食嫩笋的高表达功能家族"提供了**文献锚定的定性判据**。
本层目标是：在不依赖湿实验、不依赖本环境无法运行的 BLAST/大文件下载的前提下，
给出可复现、可审计的"笋相关性"分级，并显式声明量化连接的剩余步骤。

### 3.x.2 方法
**步骤 1 — 资源审计。** 系统梳理公开毛竹转录组（表 1），区分
（i）笋特异性资源（Peng 2013 de novo 笋转录组；MDPI 2020 'Pachyloen' 笋发育阶段转录组），
（ii）非笋对照（GA 处理 culm/seedling、rhizome、culm circRNA）。
关键事实：两个笋特异性资源均**不以 UniProt accession 为键**
（Peng 2013 为 de novo unigene；MDPI 2020 为参考基因组基因模型 `PH0100…G…`）。

**步骤 2 — 基因模型断链核验。** 抽取本课题组 253 条源蛋白中一条真实
UniProt TrEMBL 条目（A0AA96NL62）的 JSON 核验：其交叉引用仅含
EMBL / AlphaFoldDB / InterPro / Pfam 等，**不含** `PH0100…` 基因组基因模型 ID。
即"UniProt accession → 基因组基因模型"的直接映射在当前公开注释中**断链**。

**步骤 3 — 源蛋白"笋相关性"定性分级。** 对 253 条源蛋白，将其功能描述与
已发表笋高表达基因家族（Peng 2013 Fig.3：CYCA、EXP、FTK、BGL、ARF、MYB、MYC、
DOF、SAUR、AUX1、GID/GID1；MDPI 2020：转录因子、激素信号、细胞壁代谢富集）
做关键词重叠，赋予 high / medium-high / medium / low-medium / low 五级
（判定逻辑见 `scripts/phaseA/transcriptome_expression_layer.py` 的 `shoot_tier`）。
候选肽按其源蛋白集合做 set 归属（与存在性映射一致，避免首命中偏倚）。

候选池（4,950 条 2–6 aa 短肽）按源蛋白集合 set 归属：
**high 触及 65.1%**、medium-high 38.8%、medium 53.4%、low-medium 19.4%、low 3.6%。
以"最有利源蛋白"判定，**3,837 / 4,950 条候选（77.5%）其最优源蛋白属 high/medium-high 笋相关性**。

**决赛候选的源蛋白笋相关性：**
- **LPPGP** → B3VN36 细胞色素 P450 73A33（代谢酶）→ medium-high；P450 家族在幼嫩组织次生代谢中广泛表达。
- **APPSQ** → A0A3Q8AYS5 Squamosa-promoter binding protein-like（SBP 转录因子）→ high；SBP/SPL 调控茎顶端分生组织与营养生长，幼笋中高表达。
- **APQIP** → X2F5C1 MADS-box protein 4（转录因子）→ high；MADS-box 核心调控茎尖分生组织与笋发育。

### 3.x.4 诚实边界（方法学声明）
本层**未提供逐候选 TPM 定量**，根因有二，均已在方法内显式声明：
1. UniProt TrEMBL（tax 38705）条目不含 `PH0100…` 基因模型交叉引用，直接映射断链；
2. 无公开"以 UniProt 为键"的笋 TPM 矩阵；且本执行环境无 BLAST / 大文件下载能力。

剩余量化连接（逐候选笋表达量）被封装为**可一键复现的协议脚本**
`scripts/phaseA/transcriptome_join_protocol.py`，待在有 BLAST/MMseqs2 主机上：
- 路径 A：用本 253 蛋白对 de novo 笋转录组做序列比对，取 best-hit 的 identity/coverage 作为源蛋白在笋中可检出的证据；
- 路径 B：对接 MDPI 2020 基因组基因模型表达矩阵，需先由基因组 GFF 派生 accession↔PH0100 映射（同一比对亦可反向得到）。

即：**本层当前交付 = 资源审计 + 源蛋白定性笋分级（文献锚定）+ 定量连接协议（可补齐）**，
三方共同构成对"253 含非食物蛋白"软肋的透明、可复现回应。

### 3.x.5 小结
候选不仅 100% 回溯到毛竹染色体级基因组注释蛋白组（存在性），且**其源蛋白主体
（77.5% 候选）落在已发表文献报道的笋高/中高表达功能家族**（转录因子、细胞壁合成、细胞周期、激素信号）。
这把"天然存在"推进到"**可食嫩笋表达谱系内的天然存在**"，在不做湿实验的前提下，
为候选的食物转化潜力提供了计算层面的可食部位证据——与谢鹏 2026 仅报"可食用蛋白手工挑选"形成方法学对照。
