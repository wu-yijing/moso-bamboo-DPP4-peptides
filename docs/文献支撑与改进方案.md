## 3. 可直接写入论文的关键论断（带文献锚点）

1. **"毛竹（竹笋）内源蛋白确能释放 DPP4 抑制肽，且已有体外验证"** —— 引 C1（EGF IC₅₀ 366 μg/mL）与专利 CN118702773A（竹多肽 IC₅₀ 305–541 μM）。这为我们"以毛竹全蛋白组为源"的纯计算立题提供**实验层面的旁证**（注意：C1 用 PeptideRanker+CDOCKER，与我们的 iDPPIV-SCM+Vina+MM-GBSA 互补，非重复）。
2. **"DPP4 仅识别 N 端第二位为 Pro/Ala 的底物，S1 疏水口袋为首要特异性决定因素"** —— 引 B1/B2。我们的三候选（LPPGP / APPSQ / APQIP）**第二位均为 Pro（P1=Pro）**，应在结果/讨论中显式点明此"结构契合"，与 iDPPIV-SCM 中 Pro 倾向性 +0.875 相互印证，构成支持活性的强证据。
3. **"多接触 ≠ 高质量结合；对接无法区分底物与抑制剂"** —— 引 C5（原文论断）+ B1（Diprotin A 实为慢水解底物而非强抑制剂）。这正是我们用 **MM-GBSA + 接触谱（锚定催化核心 Glu146 而非糖基化偏倚）** 区分"真抑制剂 vs 假阳性"的理论依据，应在 §4.2 正面引用以强化方法论。
4. **"MM-GBSA 宜作相对排序/对接后过滤，非绝对亲和力断言"** —— 引 D2（支持用法）+ D1（BACE-1 重打分未优于 docking，提示局限）。我们的"Vina + MM-GBSA + iDPPIV-SCM 三信号联合、分歧即信息"策略本身符合文献稳健认知。
5. **"短肽（2–6 aa）是食源性 DPP4 抑制肽主流长度"** —— 引 C2/C3/C4/C8（燕麦、鹰嘴豆、藜麦、牛奶肽均为 2–6 肽）。与我们的候选长度区间一致，支撑"短肽优先"的筛选偏好。

## 4. 改进方案（按优先级）

### 方案 1（高优先·易实施）· 阶段1 引入 iDPPIV-SI 交叉验证
- **依据**：A2 独立集 ACC 98.12%，且数据集/代码开源（figshare）。
- **做法**：用 A2 的 SVM+LASSO 模型对我们的 7,988 候选重打分，与 iDPPIV-SCM 取交集/并集；在稿件中报告两模型一致性（Kappa/重叠率），弥补 SCM 的长度混杂弱点。
- **产出**：新增 `data/phaseA/idppiv_si_crosscheck.tsv` + 方法学一段。

### 方案 2（高优先·易实施）· 扩充文献比较表（含 IC₅₀ 基准）
- **依据**：C2/C3/C4/C8 提供已实验验证的食源性肽 IC₅₀。
- **做法**：向 `data/literature_dpp4_peptides.tsv` 追加 ≥10 条带实验 IC₅₀ 的肽（IPI 阳性对照、oat IPQHY、chickpea IAIPPGIPYW、quinoa WLAFR/LLPFR、soy IAVPTGVA、lupin LTFPGSAED、wheat 4 肽、bee ATHALLA、bamboo EGF），让审稿人横向比较"我们的候选 vs 已验证活性肽"。
- **注意**：纯计算候选无 IC₅₀，应在表内以"in silico only / 待验证"明确区分。

### 方案 3（高优先·论证强化）· 显式论述 P1=Pro 机制契合
- **依据**：B1/B2 + 我们候选第二位均为 Pro。
- **做法**：在 §3 结果或 §4 讨论新增 1 段，将"序列 Pro 偏好（SCM 倾向性）+ 结构 S1 口袋 Pro 特异性 + 候选 P1=Pro"三者串联，作为**支持活性的核心结构论据**。

### 方案 4（中优先·方法学提升）· MM-GBSA 协议调优 + 阴性对照
- **依据**：D3（力场/介电常数调优可显著提升）、D4（阴性对照 acetaminophen 验证区分度）。
- **做法**：
  - 对口袋限制 MM-GBSA 尝试 GB 模型 `igb=1` 或 `igb=8`（参照 D2）、内部介电常数 εin ∈ {1.0, 2.0, 4.0} 扫描；
  - 加入 1–2 个**阴性对照五肽**（随机非活性序列 / 已知弱结合肽），验证排序区分度；
  - 多次采样（如 3× 单结构独立最小化取平均）降低单结构偏差。
- **产出**：方法学小节 + 敏感性附表。

### 方案 5（中优先·前瞻）· 引入 MD 采样/结合停留时间
- **依据**：C6（wheat, tau-RaMD 停留时间）、C4（quinoa, 100 ns MD 验证）。
- **做法**：对 Top-2（LPPGP/APPSQ）跑 50–100 ns MD（本机算力需评估），以轨迹平均 MM-GBSA 替代单结构近似，并报告结合自由能随时间的 RMSD/RMSF 稳定性。
- **风险**：本环境前台 2 分钟被杀、后台任务被回收，需拆分为短片段或离线批处理；**可行性待评估**，列为 future work。

### 方案 6（低优先·边界防御）· 正面引用 MM-GBSA 局限以防御审稿质疑
- **依据**：D1（BACE-1 重打分未优于 docking）。
- **做法**：在 §4.2 明确："我们知悉 MM-GBSA 重打分未必优于 docking（D1），故将其定位为三信号联合中的相对排序指标而非独立断言，并据此解读 APQIP 的分歧。"

### 方案 7（低优先·对标）· 与毛竹同类研究（C1）做方法对比表
- **依据**：C1 是最直接先验（同源 organism、同靶点），但方法不同（PeptideRanker+CDOCKER vs iDPPIV-SCM+Vina+MM-GBSA+接触指纹）。
- **做法**：新增一小节/表，对比两种工作流的输入（8 条手工选蛋白 vs 253 条 UniProt 全蛋白组）、筛选器、对接工具、验证方式（C1 有湿实验 vs 我们纯计算），凸显我们"组学规模 + 多信号联合 + 接触指纹"的增量贡献。

### 方案 8（前瞻·路线）· 给出可落地的湿实验验证路线
- **依据**：C1（EGF IC₅₀）、C6（wheat 完整范式）、C3（Caco-2 细胞）。
- **做法**：在结论/展望写明："若具备湿实验条件，建议按 C6 范式：化学合成 LPPGP/APPSQ → DPP4 酶抑制 IC₅₀ 测定 → Caco-2 摄取/稳定性 → MD 机制解析"，与 C1 的毛竹 EGF 结果形成对标。
