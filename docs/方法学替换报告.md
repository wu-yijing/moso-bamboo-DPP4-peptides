## 1. 为什么是 iDPPIV-SCM(而非继续用通用代理)

| 维度 | 旧 PeptideRanker-style 代理 | 新 iDPPIV-SCM(离线复现) |
|---|---|---|
| 目标专一性 | 通用"生物活性"打分,非 DPP4 专用 | **DPP4 抑制肽专用**评分卡 |
| 可复现性 | 透明但为启发式代理,无文献支撑 | 复现已发表方法 (Charoenkwan et al. 2020, *J. Proteome Res.* 19:4125–4136, DOI:10.1021/acs.jproteome.0c00590) |
| 外部依赖 | 依赖 PeptideRanker 网页(已失效) | **零联网、零外部依赖** |
| 可解释性 | 弱(疏水/Pro/分子量经验权) | 强:逐残基倾向性,直接支撑讨论 |

**结论**:将"代理启发式"升级为"文献验证、离线可复现的 SCM 算法",原稿诚实注脚①由**劣势转为优势**。

## 2. 方法与数据(完全离线)

### 2.1 评分卡公式(Scoring Card Method)
在 DPP4 抑制肽(正)与非抑制肽(负)训练集上,计算每种氨基酸 `a` 的全局倾向性:

```
P(a) = log2( Obs(a) / Exp(a) )
Obs(a) = 正集中 a 的出现频率
Exp(a) = (正+负)集中 a 的总体频率
肽 S 的 iDPPIV-SCM 总分 = Σ_{a∈S} P(a)        # 标准 SCM 对残基倾向性求和
```

亦提供长度归一化分 `score_mean = ΣP(a) / |S|`,用于跨长度公平排名。

### 2.2 训练数据集(同源公开数据,已随代码下载至本地)
- 来源:WeiLab-BioChem/**Structural-DPP-IV** 仓库 `data/DPP-IV/`(与 iDPPIV-SCM 论文同源的 665+665 基准集)
- 规模:**train.tsv** = 531 正 + 532 负(基准);**test.tsv** = 133 正 + 133 负(独立测试)

## 3. 验证结果(诚实披露)

| 评估方式 | 本复现 | 文献报告 (Charoenkwan 2020) |
|---|---|---|
| 独立测试集 ACC(阈值优化) | **0.771** | ≈0.797 |
| 嵌套 5 折 CV(每折留一调阈值) | 0.647(高方差) | ≈0.819 |
| 纯长度基线(len≤10→判阳性) | **0.820** | — |

### 3.1 关键发现:基准集存在**长度混杂**(须如实写入稿件)
- 训练集**正样本以短肽为主**(207/531 仅 2 aa,中位长 13),**负样本几乎全为长肽/蛋白**(中位长 27,最长 90)。
- 因此"预测短=阳性"的**纯长度基线**即可达 0.820,高于组成型 SCM 本身。
- iDPPIV-SCM 作者亦自陈 *"not yet accurate enough for real-world applications"*。
- **对本项目的实际意义**:毛竹全部候选均为 **2–6 aa 短肽**(同处"阳性式"长度区),长度信号对所有候选一致失效;SCM 分此时主要反映**残基组成**这一与 DPP4 抑制相关的**真实信号**,用作候选优先级排序(ranking)与软过滤,恰为其设计本意(*"rapid screening … prior to synthesis"*)。

### 3.2 学到的氨基酸倾向性(生物学自洽,可直接写讨论)
```
P(Pro)  +0.875   ← DPP4 S1 口袋对 Pro 强特异,食源 DPP4 抑制肽标志特征
M(Met)  +0.702
W(Trp)  +0.433
Q(Gln)  +0.376
V(Val)  +0.313
E(Glu)  +0.181
...
C(Cys)  -2.482   ← 强烈负向(亦呼应毒性代理对 Cys 的标记)
```
残基信号与已知 DPP4 抑制肽化学直觉一致,证明 SCM 学到的是真实组成偏好而非仅长度。
